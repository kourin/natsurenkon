--------------------------------------------------------------
--------------------内容--------------------------------------

　　　　　　　どうにもイマイチな出来です。

修造れんこ　：松岡修造の名言をちりばめた音MAD好き用差分です。松岡さんごめんなさい。
--------------------------------------------------------------
--------------------使い方------------------------------------

1.つんでれんこの入ってるフォルダを開く
2.「setting」フォルダ中の「message」ファイルを安全な場所に移動
3.当Zip内の好きな方のファイルをsettingフォルダに移動、タイトルをmessageに変更
