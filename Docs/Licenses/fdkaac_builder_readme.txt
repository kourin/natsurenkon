> これは何？
nu774氏のfdkaacを自動でビルドするスクリプトです。
fdkaac_autobuildをベースにしています。
(大幅な変更があると厳しいかも)

> 使い方
builder.batを実行してください。
成功すればfdkaac.exeがビルドされるはずです。
