概要

Minimal DirectShow Player は、UI を持たない真に最小限の Windows 用メディアプレーヤです。

動作環境

必須環境

OS
Windows XP 以降

実行方法

mdsplayc <filename>
mdsplayg <filename>

mdsplayc はコンソールアプリケーションです。mdsplayg は GUI アプリケーションです。

Minimal DirectShow Player は UI を持たないため、一時停止、早送り、巻き戻しなどの操作は一切できません。映像を持つメディアの場合は再生ウィンドウを閉じることによって再生を停止し、アプリケーションを終了することができます。
